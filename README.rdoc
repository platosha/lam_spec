Шаблон спецпроекта для Look At Me

  git clone git://github.com/macovsky/lam_spec.git
  rails new APP_NAME --skip-bundle -d mysql --skip-test-unit -m lam_spec/template.rb
  
В качестве имени приложения правильно выбрать его папку на http://specials.lookatme.ru.
  
После создания приложения необходимо поправить config/lam_auth.yml и app/views/_shared/_sharing_includes.html.erb
